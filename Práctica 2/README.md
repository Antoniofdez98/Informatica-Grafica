# Práctica 2

Compilar y Ejecutar - make

Presionando Teclas:
 - 'Q' Sale del ejecutable
 - '1' Dibuja los puntos
 - '2' Dibuja las líneas
 - '3' Dibuja en un color solido
 - '4' Dibuja en colores
 - 'P' Dibuja la Pirámide
 - 'C' Dibuja el Cubo
 - 'O' Dibuja el objeto PLY
 - 'R' Dibuja el objeto Rotación
 - 'X' Dibuja el objeto Extrusión
 - 'L' Dibuja el Cilindro
 - 'N' Dibuja el Cono
 - 'E' Dibuja la Esfera
